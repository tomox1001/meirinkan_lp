# 明倫館（めいりんかん） 公式サイト

## プロジェクト概要
剣道クラブ「明倫館」の公式ホームページ。
2021年設立。「礼にはじまり礼におわる」を理念に、礼儀と思いやりの心を育む剣道クラブ。

### 主要ターゲット
- **第一ターゲット**: 子どもに剣道を習わせたい保護者（30〜40代の母親が中心）
- **第二ターゲット**: 剣道に興味を持った小学生本人
- スマホからの閲覧が大半を占めるため、モバイルファーストで設計する

---

## デザイン方針

### コンセプト
「凛とした和の空気感」×「親しみやすさ」
- 剣道の格式を感じさせつつ、小学生のクラブらしい温かみと明るさを両立する
- 保護者が安心感を持てる、清潔で信頼感のあるデザイン
- 写真がないため、イラストやアイコン、タイポグラフィで世界観を表現する

### カラーパレット
- メイン: 藍色 `#1B3A5C`（剣道着の色をイメージ）
- アクセント: 朱色 `#C53D43`（日本の伝統色、CTAに使用）
- 背景: オフホワイト `#FAFAF5`（和紙のような温かみ）
- テキスト: ダークグレー `#2D2D2D`
- サブ背景: 淡い藍 `#EDF2F7`

### タイポグラフィ
- 見出し: Noto Serif JP（明朝体で和の雰囲気）
- 本文: Noto Sans JP（ゴシック体で読みやすさ確保）
- キャッチコピー「礼にはじまり 礼におわる」は特に大きく印象的に

### 写真素材について
- 現時点では写真素材がないため、写真が入るべき箇所にはすべて**プレースホルダー（ブランク領域）**を設置する
- プレースホルダーは藍色の淡い背景（#EDF2F7）+ カメラアイコン（SVG）+ 「写真準備中」テキストで統一
- プレースホルダーのアスペクト比は 16:9（横長）を基本とする
- 将来 `public/images/` に画像を置いて差し替えるだけで済む構造にする（Next.js Image コンポーネント使用）
- プレースホルダーは共通コンポーネント `PhotoPlaceholder.tsx` として作成し、alt文言とサイズをpropsで渡せるようにする

#### 写真を入れるべき箇所（ブランク設置箇所）
1. **ヒーローセクション背景** — 稽古風景の写真（全幅、高さ 60vh 程度、オーバーレイ付き）
2. **明倫館についてセクション** — 集合写真または稽古中の写真（1枚、横幅50%程度）
3. **こんな子を歓迎セクション** — 子どもたちが稽古している写真（1枚、セクション背景またはインライン）
4. **稽古案内セクション** — 稽古場所の外観または内観の写真（1枚）
5. **体験のご案内セクション** — 体験稽古の様子の写真（1枚）

### デザインディテール
- セクション間に十分な余白をとる（最低 py-20）
- 角丸は控えめに（rounded-lg まで）
- シャドウは繊細に（shadow-sm〜shadow-md）
- 和を感じさせる装飾（細い横線、余白の美）を取り入れる
- アニメーションは上品に（フェードイン程度、派手な動きは不要）
- 写真なしの状態でもデザインとして成立するよう、プレースホルダーは美しく見せる

## ページ構成（シングルページ）

保護者がスマホでサッと見ることを想定し、1ページ完結型で設計する。

1. **ヒーローセクション** — キャッチコピー「礼にはじまり 礼におわる」+ 説明 + 体験申込みCTA
2. **明倫館について** — クラブ紹介文 + 「強く・正しく・美しい」の3つの柱
3. **こんな子を歓迎** — 4つのきっかけをカードまたはアイコンで表現
4. **稽古案内** — 日時・場所・対象をシンプルに
5. **体験のご案内** — 体験の流れ + メール連絡先 + CTA
6. **フッター** — 連絡先・コピーライト

## 技術スタック
- Next.js 15 (App Router)
- Tailwind CSS v4
- TypeScript
- Google Fonts (Noto Serif JP, Noto Sans JP)
- デプロイ先: Vercel

## コーディングルール
- コンポーネントは `app/components/` に配置
- セクションごとにコンポーネントを分割（Hero.tsx, About.tsx, Welcome.tsx, Practice.tsx, Trial.tsx, Footer.tsx）
- 日本語コンテンツはコンポーネント内にハードコード（CMS不要）
- 画像は `/public/images/` に配置（現時点では空。写真追加時にここに格納する）
- 写真が入るべき箇所には共通の `PhotoPlaceholder` コンポーネントを使う
- レスポンシブはモバイルファースト（sm → md → lg の順で記述）
- セマンティックHTML + アクセシビリティ（適切なaria属性、alt属性）
- ページ内スムーズスクロールでナビゲーション

## 実装の注意点
- 写真素材は後日追加予定。すべての写真箇所に PhotoPlaceholder コンポーネントでブランクを設置する
- ブランクはデザインの一部として美しく見せる（灰色の空ボックスではなく、藍色系の上品なプレースホルダー）
- 写真追加時は PhotoPlaceholder を Next.js Image に差し替えるだけで済むようにする
- 「礼にはじまり 礼におわる」のキャッチコピーは、ヒーローセクションで最も印象的に見せる
- CTAボタンは朱色（#C53D43）で目立たせ、「体験のお申し込み」に誘導
- メールアドレスはmailtoリンクにする
- ふりがなが必要な箇所（子ども向け）は <ruby> タグで対応可能にしておく
